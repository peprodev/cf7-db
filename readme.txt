== Description ==

### **Save Contact From 7 Submission into database and filter / view / print / export saved data as CSV, Excel Spreadsheet and ...**


This plugin lets you to browse auto-saved contact form 7 submissions and export them.
Here are some awesome features of CF7-DB:


- Zero-Configuration
- Automatically records submissions
- Filter by Contact Forms
- Auto-generated contact from field as table column
- Export as Excel
- Export as CSV
- Export as HTML
- Export as TXT
- Export as Print
